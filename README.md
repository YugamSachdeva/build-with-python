# Build With Python 🚀

This repository represents my journey of building real-world Python projects.
It is focused on applying Python concepts to practical use cases and improving problem-solving skills through hands-on development.

All the projects in this repository are created during learning and practice, keeping the code simple, structured, and easy to understand.

---

## 🚀 What This Repository Includes

This repository contains multiple Python mini-projects that cover different concepts such as:

* Logic building and problem solving
* File handling and automation
* Working with APIs
* Using external Python libraries
* Real-world project implementation

---

## 📂 Projects Overview

* **calculator/**
  A simple calculator using `match-case` and exception handling

* **file-organizer/**
  Automatically organizes files into folders based on type

* **qr-generator/**
  Generates QR codes from text or URLs

* **drink-water-reminder/**
  Sends reminders at regular intervals

* **pdf-merger/**
  Merges multiple PDF files into one

* **news-app/**
  Fetches latest news using an API

* **millionaire-game/**
  A quiz-based game inspired by "Who Wants to Be a Millionaire"

* **virtual-assistant/** *(coming soon)*
  AI-based assistant using voice commands

## ▶️ How to Run

1. Clone the repository:

   ```
   git clone https://github.com/your-username/build-with-python.git
   ```

2. Navigate to the project folder:

   ```
   cd build-with-python
   ```

3. Run any project:

   ```
   python filename.py
   ```
